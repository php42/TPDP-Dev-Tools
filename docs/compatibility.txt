YnK: gn_dat6/doll/Compatibility.csv
this is a text file that defines the type chart ( see http://en.tpdpwiki.net/wiki/Type_Chart )
0 = immune, 1 = not effective, 2 = neutral, 4 = super effective

elements appear in the following order:
Null, Void, Fire, Water, Nature, Earth, Steel, Wind, Electric, Light, Dark, Nether, Poison, Fight, Illusion, Sound, Dream, Warped

Note that NULL is used as a second type for single-type puppets, don't touch it unless you want insanity to ensue
